## Installation

- Clone this repo and `cd` into it
- Install [jaq](https://github.com/01mf02/jaq)
- Run `composer install`
- Run `npm install`
- Run `npm run dev` and start coding

## Cheatsheet

| Command                    | Description                           |
| -------------------------- | ------------------------------------- |
| composer outdated --direct | list outdated direct PHP dependencies |
| composer update            | upgrade PHP dependencies              |
| ncu                        | list outdated JS dependencies         |
| ncu -u                     | upgrade JS dependencies               |
